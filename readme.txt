Easy integration of giphy.com gifs

== Description ==

Once activated the plugin adds Giphy as provider for embed WordPress.

= Features =

* No option. Just activate the plugin and copy/paste your favorite gifs from https://giphy.com
* gutenberg block

[Follow me on Twitter](http://twitter.com/intent/user?screen_name=tweetpressfr)

== Installation ==

1. Upload plugin files to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Enjoy giphy ^^

e.g copy paste this URL in visual editor : http://giphy.com/gifs/black-girls-rock-kclxoRdCys3Mk
